# BonziCONNECT
A app that lets you controll Bonzi from Python using C#

# Usage
Run the **BonziCONNECTL.exe** in the **BonziCONNECT** folder and use this code in python :
```python
import BonziSEND

BonziSEND.speak("Hello")
BonziSEND.play("Read")
```

This python file will need to have the **BonziSEND.py** file in the same directory.

## Speak
Use :
```python
BonziSEND.speak("Hello from Python")
```

## Play animation
Use :
```python
BonziSEND.play("Wave")
```
